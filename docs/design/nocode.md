- 编辑器：针对用户友好的交互模式，进行一个汇总的 config（采用 JSON Schema 数据格式作为基础，可以进行扩充）的生成。

- 组件库：进行不同组件的开发(采用 webpack 的模块联邦)。

- 渲染器：根据编辑器生成的 config，以及组件库，渲染出页面，可以进行服务端渲染和客户端渲染的配置。

- 静态资源管理：管理图片、音频、视频，以及静态文本。
